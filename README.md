# Portofolio — Citra Sari

Portofolio berisi CV dan contoh hasil pekerjaan administrasi dan data entry untuk keperluan profesional.

## Ringkasan
Repositori ini ditujukan untuk menampilkan:
- CV profesional (format PDF/DOCX).
- Contoh pekerjaan administrasi dan data entry (tabel, laporan, template).
- Presentasi utama: Portfolio-Citra-Sari.ppt

## Struktur yang disarankan
- /cv/          — file CV terpisah (PDF / DOCX)
- /pdf/         — versi PDF dari presentasi (lebih mudah ditampilkan di GitHub)
- /assets/      — screenshot slide, preview, atau logo (PNG/JPG)
- /samples/     — contoh hasil kerja (CSV, XLSX, DOCX, PDF)
- Portfolio-Citra-Sari.ppt — file presentasi utama

> Catatan: Saat ini file .ppt seringkali tidak ter-preview di GitHub. Sangat disarankan menambahkan versi PDF di folder `pdf/` dan screenshot di `assets/` agar pengunjung dapat melihat tanpa mengunduh.

## Cara melihat
1. Jika ada `pdf/Portfolio-Citra-Sari.pdf` — buka langsung diGitHub.
2. Jika hanya ada `.ppt`:
   - Unduh lalu buka di Microsoft PowerPoint, LibreOffice Impress, atau unggah ke Google Slides.
3. Untuk melihat cepat, tambahkan 1–3 screenshot di `assets/` dan sertakan badge/image di README.
